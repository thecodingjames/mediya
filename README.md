# Mediya

A lightweight front-end app to play media streams

- Supports FireStick and Nvidia Shield controller input


## Soft-fork

```
git add remote upstream https://github.com/thecodingjames/mediya.git
```

```
git merge upstream/main --allow-unrelated-histories
```

Handle conflicts if any or keep fork's specific file changes
```
git checkout HEAD -- channels.json
```

Then commit and push
